# Prayer Requests App

This is a mobile app that I created to allow users to create and share prayer requests. Users can create and join prayer groups where they can post and share their prayer requests. This repository is for the frontend of this app is created using React Native and Expo. The backend of this app is using Java, Spring, and PostgreSQL. <br/>
See the backend repository here - <a href="https://github.com/kdu2030/prayerappservice" target="_blank" rel="noreferrer">Backend repository</a>

# Features
* User sign up and sign in
* i18n Support for English and Chinese
* Creating Prayer Groups - Allows users to add a description, rules, and upload an avatar or banner image for the prayer group
* Searching for prayer groups
* Public and private prayer groups - Private means that a user has to be approved by an administrator before joining a prayer group
* Adding prayer requests
* Adding comments to prayer requests

# Features Under Development
* Editing, deleting, and bookmarking prayer requests
* Home page feed, which shows prayer requests from the groups that the user has joined
* Focus mode - Which prevents users from going to other apps while a prayer session is active
* User settings - Allows users to upload a profile picture, change their user credentials
